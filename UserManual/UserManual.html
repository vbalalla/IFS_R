 1.0 INTRODUCTION

This is the user guide, documented to give understanding of its features to all the users who are having privileges to use the ‘CVs and Recruitment Management System’ which is developed for IFS. Hence this explains how the system works and gives an opportunity to learn about all its functionalities.

1.1 System Overview

CV and Recruitment Management System is a web based CV management system which helps to ease the tracking candidates functionality and further recruitment process functionalities.

Initially this system will get bunch of CVs in different formats such as doc files, PDF files and so on, as the input. System will then extract necessary,basic  data from the CVs. If some of the details were failed to be extracted, those will be filled manually. Filtering of the CVs with threshold waiting period issue, will happen for CVs through the system automatically.Responsible officers are allowed to select the best candidates for interviews manually. Those who would be short listed will be called for interviews.

Interview scheduling is also facilitated by the system. The receptionist should manually contact the interview panel members as the way how she is doing it currently to get the convenient time slot and then she has to call the candidates and schedule the interviews accordingly where she is given the chance to enter it in front of each candidate and then automatic emails informing that would be sent to him.

During the interviews, each and every interview panel members are given a chance to enter marks for a candidate parallelly. Then the system total the candidates’ marks and rank them. Again, selecting candidates for next interviews, scheduling interviews and interviewing process flow same as above until the company hire the best workforce to the company.

Information available in the system are protected and continuously updated for future reference which would automatically give the ability to generate statistical reports on demand of different users.

A user guide, also commonly known as a manual, is a technical communication document intended to give assistance to people using a particular system. The language used here is matched to the intended audience, with jargon kept to a minimum and explained thoroughly (Wikipedia, 2015). 2.0 SYSTEM SUMMARY

System Summary section provides a general overview of the system. The summary outlines the uses of the system’s hardware and software requirements, system’s configuration, user access levels and system’s behavior in case of any contingencies.

2.1 System Configuration

  • Cv and Recruitment Management System (Resume Trekker) is a web based system which is specifically designed to run on the server of IFS R And D International (pvt) Ltd.
  • System needs to be incorporated with an operating system such as Windows,Mac OS and Linux.
  • Server should be run on an web browser that supports HTML5 and CSS3,therefore system supports Firefox 3.5 or higher versions,Google Chrome, Internet Explorer  or higher versions(including microsoft edge). (Blog.deepbluesky.com,2015)
  • System automatically configures the database once the system is connected to the server which stores all the necessary data regarding the system.
  • Since the system is supposed to send emails as an additional feature ,it would be better if it is connected to the internet even though it is optional.
  • Once the system is successfully installed and connected to the server ,system can be run without any further configurations.

When search execute, if results are available, search results will be listed down. If there are no results available, system will show “Sorry there are no results available for this search”.

4.6 Change Threshold Period 

User who is logged in as administrative officer is allowed to click on the “change threshold period” button.Then he would be redirected to this page.

There in front of years,he has to enter the number or he can press on the up or down arrow which would increase the value by 0.1 decimals.

Then he needs to press the “save” or “cancel” button.

If cancel button is clicked,entered value regarding threshold period would be reset.

If save is selected,since the threshold period is a critical factor,a warning message will be prompted to user.

User can select default interview panels and can add members to panel or delete added members by clicking “Add member” and “Edit member” buttons.The status of the panel can be set to default indicating that this is the default interview panel which exists in the system and all the other new creating interviews will load this default panel.
